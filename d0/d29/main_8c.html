Detailed Description

Glavni program za RUS_Lab1_prekidi (verzija za predaju profesoru).

Ovaj modul povezuje sve ostale module:

  • prekidi tipkala (buttons)
  • hardverski timer (timer)
  • ultrazvučni senzor HC-SR04 (hcsr04)
  • Serial Plotter vizualizacija (plotter)
  • centralna inicijalizacija prekida (interrupts)

Napomena:

  • U Arduino okruženju stvarni main() generira core, a korisnik definira samo setup() i loop().
  • Ovaj main.c služi kao čista, modularna i dokumentirana prezentacija projekta za potrebe kolegija.
