\# RUS Lab1 — Višestruki prekidi



Ovaj projekt implementira sustav s višestrukim prekidima:

\- prekid tipkala (visoki prioritet)

\- timer prekid (srednji prioritet)

\- HC-SR04 senzor (niski prioritet)

\- UART RX prekid (najniži prioritet)



\## Struktura projekta

\- `src/` — izvorni kod

\- `wokwi/` — Wokwi simulacija

\- `diagrams/` — CFG i dijagrami

\- `docs/` — automatski generirana dokumentacija (Doxygen)

\- `.github/workflows/` — GitHub Actions



\## Dokumentacija

Automatski generirana dokumentacija dostupna je u `docs/html/index.html` ili preko GitHub Pages.



\## Wokwi simulacija

Datoteke se nalaze u `wokwi/` direktoriju.
